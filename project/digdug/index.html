DigDug - Redesign

Project: Redesign an old game.
Team: Team project: 3 artists, 3 programmers and 3 designers.
Date: November-January 2015-2016, 8 week project
Tasks: Design, Model, Texture, Rig and Animate the two enemy characters.
Programs: Maya, Photoshop, Mudbox, XNormal.
Description: Re-invent a retro game. We decided on DigDug as our retro game. In this game you have to kill the enemies by shooting them or digging away the ground on which they stand. We mainly updated the visuals but also completely removed the shooting mechanic adding a puzzle element to the game. This was my first time rigging and animating some characters, which I also had to redesign and make myself. This project also was my first experience working on a game in a team.

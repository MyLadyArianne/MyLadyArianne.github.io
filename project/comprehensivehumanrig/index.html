Comprehensive Human Rig

Project: Create a human character rig.
Team: Individual project
Date: March-April 2017, 8 week project.
Tasks: Created the rig.
Programs: Maya.
Description: I wanted to explore rigging and decided to start with learning how to rig a character in a more advanced way. I tried to put as many options as possible into this rig. The character is also able to hold on to a weapon so it could be used for a Viking like character.

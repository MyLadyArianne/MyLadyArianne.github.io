Ragnarok

Project: GDC worthy game prototype for a external client.
Team: Team project with 16-20 people.
Date: September 2018 - June 2018, 4 times 8 weeks project.
Tasks: Rigging and Animation.
Programs: Maya, Unreal.
Description: Ragnarok is a turn based tactical RPG. The game takes place on an alien planet were humans came to mine resources, but now the aliens are fighting back. You have to guide your small team of Mech warriors through the level to take back settlements and make sure the resource can continued to be mined by defeating the opposition. This project is being made as a demo for a client whom would like to get funding for this project.

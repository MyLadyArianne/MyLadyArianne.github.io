Raptor rig

Project: Create a rig where a lot of the secondary motions are automated.
Team: Individual project.
Date: September 2018 - Now.
Tasks: Rigging
Programs: Maya, Z-Brush, Substance Painter 2
Description: THIS PROJECT IS A WORK IN PROGRESS. Eventually you should be able to use this rig to get the base animation in very quick. Than be able to add all the values down to the 'normal' rig control layer and continue tweaking the animation from there.
